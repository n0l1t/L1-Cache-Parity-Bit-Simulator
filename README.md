# L1 Cache Parity Bit Simulator (SwiftUI)

Программная модель поведения кэш-памяти процессора первого уровня (L1 Cache), предназначенная для исследования эффективности механизма защиты данных с использованием бита чётности при воздействии статических помех.
Проект разработан в виде macOS-приложения с использованием SwiftUI и Charts и используется как вычислительный эксперимент в рамках курсовой работы

## Описание модели

Модель реализует имитационную симуляцию строки кэш-памяти фиксированной длины.
Основные параметры моделирования:
* длина строки кэша: 32 бита
* вероятность ошибки: p
* число итераций: до 50 000
* механизм контроля: even parity
Каждая итерация симуляции является независимым экспериментом и включает:
1. создание новой строки кэш-памяти;
2. вычисление бита чётности;
3. внесение ошибок;
4. проверку целостности данных;
5. сбор статистики.

## Типы моделируемых ошибок

#### Одиночные ошибки (solo)
* инвертируется один случайно выбранный бит;
* моделирует одиночные сбои ячеек памяти (SEU);
* обнаруживается битом чётности в 100% случаев.

#### Множественные ошибки (multiple)
* каждый бит независимо инвертируется с вероятностью p;
* моделирует распределённые статические помехи;
* часть ошибок не обнаруживается из-за чётного числа инверсий.
  
## Приложение отображает результаты моделирования в виде:

* столбчатой диаграммы (обнаруженные vs необнаруженные ошибки);
* гистограммы распределения ошибок по битам строки кэша, позволяющей оценить равномерность воздействия помех.
	
Для визуализации используется фреймворк Charts.
